# language-specs — ILM/Hindawi language specifications

Token-equivalence specifications: a localized language is a lexical form of a host language (C, C++, Python,
...) by mapping keywords/tokens; **semantics are the host language's**. The reference example is **HindiC++**
in `hindawiai/chintamani` (an ISO-style spec). This repo collects specs and a template.
